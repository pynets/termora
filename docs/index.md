# Termora

A modern, high-performance cross-platform desktop developer toolbox.

[Get started](getting-started.md){ .md-button .md-button--primary }
[View on GitHub](https://github.com/pynets/termora){ .md-button }

Termora brings the tools developers use every day into one focused desktop app: terminal sessions, remote files, databases, technical notes, and screenshot references.

## One toolbox, four workflows

<div class="grid cards" markdown>

-   :material-console-line: **Terminal & SSH**

    Connect to multiple SSH hosts, manage SFTP files, and keep useful command snippets close at hand.

    [Explore terminal features](features/terminal.md)

-   :material-database-search: **Database explorer**

    Inspect SQLite and PostgreSQL data, write SQL, and browse results without leaving Termora.

    [Explore database features](features/database.md)

-   :material-notebook-edit: **Notes & LaTeX**

    Write Markdown notes with live preview, tables, code blocks, formulas, and PDF export.

    [Explore notes features](features/notes.md)

-   :material-image-multiple: **Screenshot pinning**

    Capture, annotate, and pin a reference image above other windows while you work.

    [Explore screenshot features](features/screenshot.md)

</div>

!!! tip
    Termora is local-first. Your hosts, notes, and database connections stay on your machine unless you explicitly move them elsewhere.
