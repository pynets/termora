# Screenshot pinning

Capture a visual reference, make a quick annotation, and keep it visible while you code.

![Screenshot tools](../screenshots/remote.png){ .feature-screenshot }

## Highlights

- Global shortcut capture with `⌥+Shift+X`
- Crop, draw, and highlight captured images
- Always-on-top floating pin windows
- Useful for comparing UI states, logs, and design references

This workflow keeps temporary visual references close without adding another heavy desktop utility.
