# Notes & LaTeX

Notes is a local Markdown workspace for documentation, scratch work, and technical references.

![Notes workspace](../screenshots/notes.png){ .feature-screenshot }

## Highlights

- Live Markdown preview
- Tables, task lists, code blocks, and headings
- LaTeX math rendering with `$$ ... $$`
- Outline navigation and find-in-note support
- PDF export for sharing or archiving

Notes are stored locally, making the feature useful even when you are working offline or with sensitive technical material.
