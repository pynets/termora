# Database explorer

The database workspace gives you a compact SQL workflow without requiring a separate client for every engine.

![Database explorer](../screenshots/database.png){ .feature-screenshot }

## Highlights

- SQLite and PostgreSQL connections
- SQL editor with syntax highlighting
- Table and column browsing
- Query results in a structured data view
- Local connection storage

The database layer is designed for quick inspection and focused development rather than replacing a full production database administration suite.
