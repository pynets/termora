# Terminal & SSH

Termora combines an interactive terminal with a visual remote-file browser, so connecting to a server and moving files can happen in the same workspace.

![Terminal workspace](../screenshots/terminal.png){ .feature-screenshot }

## Highlights

- Multiple SSH sessions with tabs and split panes
- SFTP browsing with drag-and-drop file operations
- Command snippets for frequently used commands
- Configurable link matching and terminal highlighting
- Local session logging for troubleshooting

!!! tip
    Keep a small set of safe, repeatable commands in snippets so routine server work stays consistent.
