# Termora

现代、高效、跨平台的桌面开发工具箱。

[快速开始](getting-started.md){ .md-button .md-button--primary }
[访问 GitHub](https://github.com/pynets/termora){ .md-button }

Termora 将开发者日常使用的工具集中到一个专注的桌面应用中：终端会话、远程文件、数据库、技术笔记和截图参考。

## 一个工具箱，覆盖四类工作流

<div class="grid cards" markdown>

-   :material-console-line: **终端与 SSH**

    连接多个 SSH 主机、管理 SFTP 文件，并随时使用常用命令片段。

    [查看终端功能](features/terminal.md)

-   :material-database-search: **数据库客户端**

    查看 SQLite 与 PostgreSQL 数据，编写 SQL，并直接浏览查询结果。

    [查看数据库功能](features/database.md)

-   :material-notebook-edit: **笔记与 LaTeX**

    使用 Markdown 编写技术笔记，支持实时预览、表格、代码、公式和 PDF 导出。

    [查看笔记功能](features/notes.md)

-   :material-image-multiple: **截图贴图**

    截图、标注并将参考图片置顶到其他窗口之上，方便边开发边对照。

    [查看截图功能](features/screenshot.md)

</div>

## 为桌面而生

| | Termora |
| --- | --- |
| 框架 | Flutter |
| 支持平台 | macOS、Windows、Linux |
| 存储方式 | 本地优先的 SQLite 与偏好设置 |
| 开源协议 | MIT |

!!! tip
    Termora 采用本地优先设计。主机、笔记和数据库连接都会保存在本机，除非你主动将它们复制到其他位置。
